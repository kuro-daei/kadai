# 課題「FizzBuzz」

0 から `n` までの数で、

- 3 で割り切れるなら `Fizz`
- 5 で割り切れるなら `Buzz`
- 両方で割り切れるなら `FizzBuzz`
- それ以外は数字をそのまま

と表示します。

```
0 1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz 16 17
```

`src/fizzbuzz.js` の `fizzbuzz` を変更してください。

変数 `n` まで繰り返します。答えは、配列の `numbers` に入れてください。

```
numbers[0] = 0;
numbers[1] = 1;

```

みたいな状態になります。
